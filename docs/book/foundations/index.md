# Part I — Foundations

The starting point. By the end of this Part you'll have goi installed, a "Hello World" window on screen, and a working mental model of GObject — the type system everything in GTK is built on.

If you're coming from another GUI ecosystem, read the matching **migration guide** instead of (or alongside) the linear chapters. The migration guides translate concepts you already know rather than introducing them from scratch.
