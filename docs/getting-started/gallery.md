# Gallery

The gallery path should answer a simple question early: what kinds of programs
does `ginext` help you build?

## Where to look

Start with the examples already in this repository:

- `examples/hello_template.py` for a compact template-based GTK example
- `examples/gio/` for file, settings, drive, and volume examples
- `examples/async/` for Gio and asyncio-oriented flows
- `examples/playground/` for the larger workbench-style draft app
- `examples/web_browser/` and `examples/terminal/` for more complete apps

## What this page should become

As the docs site grows, this section should link screenshots and short
descriptions to the most useful example apps, while the top-level site gallery
can stay focused on broader visual showcase material.
