# Part VIII — Shipping

How to get your app from a working dev tree to a user's machine. One Part per distribution target plus the cross-cutting concerns (translations, app metadata, CI).

The single most common reason apps stall before release: shipping is where the work multiplies. This Part is meant to be skimmed top-to-bottom and then used as a per-target reference.
